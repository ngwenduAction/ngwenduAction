## Selected Projects

### SÉRA — Interactive Skincare Campaign Microsite
A premium skincare microsite built with React, Tailwind CSS, and GSAP, focused on editorial layout, responsive media handling, motion systems, and production-conscious frontend polish.

**Highlights**
- Modular content architecture
- Reusable GSAP hooks
- Reduced-motion support
- Responsive media integration and optimization

### Signal Field — Real-Time WebGL Interactive
A shader-driven Three.js scene built with instancing, GLSL, and bloom. Scroll controls motion energy and depth, creating a reactive luminous field animated primarily on the GPU.

**Highlights**
- `InstancedMesh` architecture
- Custom GLSL shaders
- GPU-first animation design
- Procedural motion and post-processing
